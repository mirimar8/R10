# R10

## Description
A cross-platform (iOS/Android) mobile application using React Native for an upcoming developer conference called R10.

## Technologies Used
* React Native
* AsyncStorage


## Installation 

install all dependencies:
```bash
npm install
```

### Run for ios

```bash
npm run ios
```

### Run for android

```bash
npm run android
```
