## Server

Commands must be run from the `server` directory:

### Installation

```bash
npm install
```

### Run

```bash
npm start
```

## Client

Commands must be run from the `client` directory:

### Installation

```bash
npm install
```

### Run

```bash
npm run start:dev
```
